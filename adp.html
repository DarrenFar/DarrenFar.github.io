ABC Cars portal

You have been approached by ‘ABC Cars Pte Ltd’ as a website developer to develop a Used Car Sales portal. Users will be able to register in the portal using the Registration Page. Users of the portal can search for Cars using Make, Model, Registration & Price Range. Users will be able to view the Car information after searching them. The portal allow users to login, post Car for sale. Administrator will be able to manage the users, cars, bidding, and appointment. The Website will be implemented using spring boot framework which include Spring, Security, Spring Data JPA, MySQL server, documented and tested by adopting the Risk Based Testing (RBT).
